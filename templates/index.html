A Flask REST API Used to do CRUD Actions on excel sheets parsed from Wiki table


The API has 2 stages, First you upload the file generated from the wiki parser then you can access the API to do CRUD operations


Part I (Upload file)

  • This is a mandatory stage in order use the CRUD operations via the API
  • This is a POST method that can be done by CURL, Postman or any tool/app that can send HTTP responses
  • File can be uploaded via 'rooturl/upload' with a 'file' argument containing the file path

How To:

An example to upload a file via curl would be like command below bearing in mind that you need to replace (path - extension - url) with the actual values

  • curl -X POST -F "file=@path.extension" "url/upload"

Part II CRUD operations

After uploading the file all CRUD operations will be available via the /api route


Part 2.1 GET Reuqest

  • Availabe via the base url 'url/api'
  • Retrieves all table data in the excel file and return it as a JSON string
  • This is a GET method that can be done by CURL, Postman or any tool/app that can send HTTP responses

How To:

An example to make a get request via curl would be like command below bearing in mind that you need to replace (url) with the actual value

  • curl url/api

Part 2.2 POST Reuqest

  • Availabe via the base url 'url/api'
  • Appends an additional entry to table of books
  • This is a POST method that can be done by CURL, Postman or any tool/app that can send HTTP responses
  • Request should include these arguments [novel,author,country]

How To:

An example to make a post request via curl would be like command below bearing in mind that you need to replace (url - novelname - authorname - countryname) with the actual values

  • curl -X POST -d novel=novelname -d author=authorname -d country=countryname url/api

Part 2.3 PUT Reuqest

  • Availabe via the base url 'url/api'
  • Edit an existing entry to table of books using book ID or create it if ID not found
  • This is a PUT method that can be done by CURL, Postman or any tool/app that can send HTTP responses
  • Request should include these arguments [bookno,novel,author,country]

How To:

An example to make a put request via curl would be like command below bearing in mind that you need to replace (url - booknumber - novelname - authorname - countryname) with the actual values

  • curl -X PUT -d bookno=booknumber -d novel=novelname -d author=authorname -d country=countryname url/api

Part 2.3 DELETE Reuqest

  • Availabe via the base url 'url/api'
  • Delete an existing entry to table of books using book ID or do nothing if ID not found
  • This is a DELETE method that can be done by CURL, Postman or any tool/app that can send HTTP responses
  • Request should include this argument [bookno]

How To:

An example to make a delete request via curl would be like command below bearing in mind that you need to replace (url - booknumber) with the actual values

  • curl -X DELETE -d bookno=booknumber url/api
